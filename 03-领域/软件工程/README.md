# 🏗️ 软件工程

## 📖 领域定义

**这个领域是什么？**
软件工程是对工程实践、编码标准、架构设计、系统质量的持续积累。区别于「技术领导力」（关注人与团队），这里聚焦**技术本身**。

**与其他领域的边界**

| 领域 | 关注点 |
|------|--------|
| 软件工程（本领域） | 代码质量、设计模式、工程实践、技术栈 |
| 计算机从业 | CS 基础理论（算法、操作系统、网络） |
| 技术领导力 | 技术决策、团队管理、架构治理 |
