# 4-算法

## 📖 领域简介

算法与数据结构是计算机科学的核心，包含常见数据结构、算法设计技巧和复杂度分析。

## 🎯 学习重点

- 基础数据结构（数组、链表、栈、队列、树、图）
- 排序算法（快排、归并、堆排序等）
- 查找算法（二分查找、哈希表）
- 动态规划
- 贪心算法
- 回溯算法
- 图论算法（最短路径、最小生成树）
- 时间/空间复杂度分析
