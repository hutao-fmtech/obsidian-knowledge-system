# 3-计算机网络

## 📖 领域简介

计算机网络涵盖网络协议、体系结构、网络安全等内容，是现代互联网的基础。

## 🎯 学习重点

- OSI七层模型与TCP/IP四层模型
- 数据链路层协议
- IP协议与路由算法
- TCP/UDP传输层协议
- HTTP/HTTPS应用层协议
- 网络安全与加密
- DNS、CDN等基础设施
