# 5-数据库

## 📖 领域简介

数据库技术包含关系型数据库、NoSQL数据库、数据库原理和优化技术。

## 🎯 学习重点

- 关系型数据库理论（范式、ER模型）
- SQL语言与查询优化
- 事务处理（ACID特性）
- 索引原理（B树、B+树、哈希索引）
- 数据库锁机制
- MySQL/PostgreSQL实践
- NoSQL数据库（MongoDB、Redis、Cassandra）
- 分布式数据库
