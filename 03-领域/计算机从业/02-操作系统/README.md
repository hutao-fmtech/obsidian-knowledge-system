# 2-操作系统

## 📖 领域简介

操作系统是管理计算机硬件与软件资源的系统软件，包含进程管理、内存管理、文件系统等核心模块。

## 🎯 学习重点

- 进程与线程管理
- 内存管理与虚拟内存
- 文件系统设计
- I/O管理
- 死锁处理
- 进程调度算法
- Linux/Unix 内核原理
