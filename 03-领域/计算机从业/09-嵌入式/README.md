# 9-嵌入式

## 📖 领域简介

嵌入式系统涉及硬件与软件结合的底层开发，包括单片机、实时操作系统等。

## 🎯 学习重点

- 嵌入式系统架构
- 单片机编程（ARM、STM32、Arduino）
- 实时操作系统（FreeRTOS、RT-Thread）
- 嵌入式Linux开发
- 外设接口（GPIO、I2C、SPI、UART）
- 中断与定时器
- 低功耗设计
- 硬件调试技术
