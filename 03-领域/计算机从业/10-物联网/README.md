# 10-物联网

## 📖 领域简介

物联网（IoT）技术涉及智能设备、传感器网络、边缘计算等新兴领域。

## 🎯 学习重点

- IoT架构设计（感知层、网络层、应用层）
- 传感器技术与数据采集
- MQTT、CoAP等IoT协议
- 边缘计算与雾计算
- 智能家居技术
- 工业物联网（IIoT）
- IoT安全与隐私
- 设备管理平台
