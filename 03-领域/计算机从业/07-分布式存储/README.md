# 7-分布式存储

## 📖 领域简介

分布式存储涉及大规模数据存储、一致性协议、容错机制等分布式系统核心技术。

## 🎯 学习重点

- 分布式存储架构（GFS、HDFS）
- 一致性协议（Paxos、Raft）
- CAP理论与BASE理论
- 数据分片与副本
- 对象存储（S3、Ceph）
- 分布式文件系统
- 存储性能优化
- 数据一致性与可用性权衡
