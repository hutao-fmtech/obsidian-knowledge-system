# 11-无线射频

## 📖 领域简介

无线射频技术涉及电磁波传播、天线设计、射频电路等硬件层面的无线通信技术。

## 🎯 学习重点

- 电磁波理论基础
- 射频电路设计
- 天线原理与设计
- 频谱分析
- 调制解调技术
- 射频测试与调试
- SDR软件定义无线电
- RFID技术
