# 12-无线通信

## 📖 领域简介

无线通信涵盖移动通信技术、无线网络协议、通信系统设计等内容。

## 🎯 学习重点

- 移动通信技术（3G/4G/5G）
- WiFi协议栈（802.11系列）
- 蓝牙技术（BLE）
- ZigBee、LoRa等低功耗协议
- 信道编码与纠错
- 多址技术（FDMA、TDMA、CDMA）
- 通信系统仿真
- 无线网络优化
