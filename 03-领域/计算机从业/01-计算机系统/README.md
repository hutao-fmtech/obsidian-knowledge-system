# 1-计算机系统

## 📖 领域简介

计算机系统涵盖计算机体系结构、组成原理、硬件设计等核心内容。

## 🎯 学习重点

- 冯·诺依曼体系结构
- CPU 组成与工作原理
- 存储层次结构
- 指令系统设计
- 流水线与并行处理
- 总线与I/O系统

### 关键要点
- ✅ **理解"为什么"比"是什么"更重要**
- ✅ 所有设计都是权衡的结果（性能vs成本vs复杂度）
- ✅ 局部性原理是计算机系统最重要的假设
- ✅ 用数据说话，用Profiler找瓶颈
- ✅ 算法优化 > 实现优化
