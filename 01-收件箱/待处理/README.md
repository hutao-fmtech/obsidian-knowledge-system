# 📋 待处理

> 临时存放需要进一步处理的内容

---

## 📝 使用说明

这个文件夹是临时中转站，用于存放需要进一步分类和处理的内容。

### 适合放在这里的内容
- 📄 需要阅读的文档
- 📧 转发的邮件内容
- 🔗 收藏的网页链接
- 📝 会议记录待整理
- 📦 下载的资料待归档

---

## ⏰ 处理原则

### 48小时原则
**所有内容在48小时内必须处理完毕**

### 处理流程
1. **快速判断** - 这是什么？需要行动吗？
2. **立即处理** - 2分钟内能完成的立即做
3. **分类归档** - 明确去向，移至对应位置
4. **设置提醒** - 需要等待的设置跟进提醒

---

- → 删除：无价值或过时的内容
