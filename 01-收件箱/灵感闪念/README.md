# 💡 灵感闪念

> 捕捉灵光一现的想法和创意

---

## 📝 使用说明

这个文件夹用于存放突然冒出的想法、灵感和创意点子。

### 适合放在这里的内容
- ✨ 突然的创意和想法
- 💭 有趣的观点和思考
- 🔮 未来可能的项目方向
- 🎯 潜在的写作主题
- 🚀 产品/功能创意

### 不适合的内容
- ❌ 待办事项（应放入任务管理）
- ❌ 参考资料（应放入04-资源）
- ❌ 完整笔记（应放入10-卡片盒）

---

## 🔄 处理流程

### 记录
快速记录，不追求完美：
- 一句话描述核心想法
- 简单的上下文说明
- 相关的灵感来源

### 定期整理
每周回顾时处理这些灵感：

1. **值得深入** → 扩展为完整笔记，移至[[10-卡片盒-README|卡片盒]]
2. **可以执行** → 转为项目或任务，移至[[02-项目/02-项目-README|项目]]
3. **暂时孵化** → 保留在此，等待时机
4. **已过时** → 删除
