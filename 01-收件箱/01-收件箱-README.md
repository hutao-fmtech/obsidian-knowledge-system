## 🔄 使用方法

### 收集阶段
1. 阅读时的想法 → 快速笔记
2. 保存的文章/资料 → 待处理
3. 突然的灵感 → 灵感闪念

### 整理阶段 (每日晚上15分钟)
1. 逐条检查收件箱内容
2. 判断信息类型并移动:
   - 🎯 可执行任务 → [[02-项目/README|项目]]
   - 🌟 持续关注 → [[03-领域-README|领域]]
   - 📚 参考资料 → [[04-资源-README|资源]]
   - 💡 知识洞察 → [[10-卡片盒-README|卡片盒]]
   - 🗑️ 无用信息 → 删除
3. 清空收件箱

---

## 💡 最佳实践

**快速收集原则**:
- 不打断当前工作
- 记录核心内容即可
- 不在收集时整理

**整理技巧**:
- 每天必须清理
- 不让信息堆积
- 5W1H判断法
