# ⚡ 快速笔记

> 随手记录，不打断心流

---

## 📝 使用说明

这个文件夹是最轻量的捕获入口，用于记录随时冒出的临时想法。

### 适合放在这里的内容
- 📌 工作中突然想到的事情
- 🔗 随手看到的链接或关键词
- 📋 临时的备忘和提醒
- 💬 对话中值得记住的观点
- 🧩 零散但可能有用的信息片段

### 不适合的内容
- ❌ 完整的创意构想（应放入 [[01-收件箱/灵感闪念/README|灵感闪念]]）
- ❌ 需要多步处理的资料（应放入 [[01-收件箱/待处理/README|待处理]]）
- ❌ 已经成型的笔记（直接放入对应目录）

---

## 🔄 处理流程

### 记录原则
- **快**：30 秒内完成，不纠结格式
- **短**：一两句话足够，关键词优先
- **标注来源**：方便后续追溯

### 每日清理（晚上 15 分钟）

1. **有价值 + 可执行** → 移至 [[02-项目/02-项目-README|项目]]
2. **有价值 + 需深入** → 移至 [[10-卡片盒-README|卡片盒]] 或 [[03-领域-README|领域]]
3. **有价值 + 待进一步处理** → 移至 [[01-收件箱/待处理/README|待处理]]
4. **有创意** → 移至 [[01-收件箱/灵感闪念/README|灵感闪念]]
5. **已无用** → 删除

> [!warning] 24 小时原则
> 快速笔记不过夜。当天记录，当天清理。

---

## 📄 快速笔记格式

无需模板，直接写即可。建议命名：`临时想法-YYYYMMDD.md`

```markdown
# 一句话标题

- 核心内容
- 相关关键词或链接
```
