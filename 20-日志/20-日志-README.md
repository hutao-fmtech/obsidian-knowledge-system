## 🎯 快速导航

### 本周视图
```dataview
LIST
FROM "20-日志/日记"
WHERE file.cday >= date(today) - dur(7 days)
SORT file.name DESC
```

### 最近回顾
```dataview
TABLE type as "类型", file.mtime as "更新时间"
FROM "20-日志"
WHERE contains(file.name, "W") OR contains(file.name, "月记")
SORT file.mtime DESC
LIMIT 5
```

## 💡 日志写作提示

### 每日三省
1. **今日成就**：今天完成了什么？
2. **今日挑战**：遇到了什么困难？如何应对？
3. **今日感悟**：学到了什么？有什么思考？

### 周回顾框架
1. **本周亮点**：最值得记录的3件事
2. **目标进展**：各项目标完成情况
3. **时间分析**：时间都花在哪了？
4. **下周重点**：下周要聚焦什么？

### 月复盘框架
1. **数据回顾**：关键指标和数据
2. **成就清单**：完成的重要事项
3. **问题分析**：遇到的困难和原因
4. **经验教训**：学到的经验和教训
5. **下月规划**：下月的目标和计划

## 🌟 使用建议

1. **养成习惯**：每天固定时间写日记（如晚上睡前）
2. **真实记录**：诚实面对自己，记录真实感受
3. **定期回顾**：周五晚上写周记，月末写月记
4. **保持简洁**：不求长篇大论，重在坚持
5. **关联笔记**：将日志与项目、目标关联

> [!tip] 坚持的秘诀
> 不要期望每篇日记都完美，重要的是持续记录。即使只写几句话也好过不写。
